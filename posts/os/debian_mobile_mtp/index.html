Debian使用MTP和手机互传文件

背景#

手机（安卓）上拍了不少照片和视频，需要传输到电脑上进行备份、筛选、刻盘等操作，使用无线方案总感觉操作麻烦。win上用数据线连接后，可以像操作本地磁盘一样操作文件，linux上没理由无法实现啊。

解决方案#

搜索后，其实方案很简单：安装gvfs-backends和gigolo即可。

1
2

sudo apt update
sudo apt install gvfs-backends gigolo

使用方法#

  1. 用数据线连接手机和电脑

  2. 在手机上打开开发人员选项，打开USB调试，切换到传输文件或传输照片模式

  3. 在电脑菜单中找到Gigolo 远程连接，打开，找到手机，双击

使用效果#

确实能像操作本地磁盘一样操作文件，但速度比较感人，win上其实也是比较慢的。之前单纯以为是数据线的问题，其实也是有MTP协议的原因。
