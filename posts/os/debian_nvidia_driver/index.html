Debian安装Nvidia驱动

背景#

系统安装完之后，默认的显卡驱动就能好好的工作，这点还是比较满意的，但用久了之后就会发现：看个高清视频啥的，CPU使用率就会蹭蹭上去，风扇呼呼的，总感觉N卡压根就没派上用场，所以就想换上Nvidia的闭源驱动看看效果。

安装驱动#

准备工作#

  1. 修改/etc/apt/sources.list，在main后加入contrib non-free，已有则跳过

  2. 禁用BIOS中的Secure Boot（！！！非常非常非常重要！！！）

由于我一开始没有禁用Secure Boot，反复安装并修改了许多配置，浪费时间不说，原始安装能否像以下步骤那样光滑已无法考证，除非把系统也重装了。

安装#

  1. 更新包索引
1

sudo apt update
  1. 安装nvidia-detect
1

sudo apt install nvidia-detect

  1. 运行nvidia-detect，我的提示安装nvidia-driver即可

  2. 安装nvidia-driver

1

sudo apt install nvidia-driver
  1. 重启

双卡切换#

Debian官方仓库中并没有nvidia-prime，所以使用envycontrol来实现。

  1. 下载envycontrol

  2. 安装

1

sudo apt install ./python3-envycontrol_3.5.1-1_all.deb

  1. 查看当前显卡模式sudo envycontrol --query

  2. 切换显卡

1
2
3

sudo envycontrol -s integrated # 切换到 集显
sudo envycontrol -s hybrid     # 切换到 混合模式
sudo envycontrol -s nvidia     # 切换到 N卡
  1. 重启
