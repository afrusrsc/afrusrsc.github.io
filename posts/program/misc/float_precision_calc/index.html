a = 1 / 10;       //0.1
b = 1.0 / 10;     //0.1
c = 1 / 10.0;     //0.1
d = 1 - 9.0 / 10; //0.09999999999999998
e = 1 - 9 / 10;   //0.09999999999999998

从以上可看出：

  1. Delphi和python 3，整数/整数=浮点数；java、Go、C#，整数/整数=整数。

  2. Delphi、Go、C#(.Net Framework)比java、C#(.NET Core)、python 3的浮点数精度要高。

  3. Delphi按自然认知书写即可；Go、C#(.Net Framework)需要注意一下书写语法；python 3浮点数做比较时要做下特殊处理才行；java、C#(.NET Core)不仅要注意书写语法，比较时还要做特殊处理。

综合精度和方便度来看，Delphi才是王者，Go、C#(.Net Framework)次之，python 3再次，java、C#(.NET Core)末之。
